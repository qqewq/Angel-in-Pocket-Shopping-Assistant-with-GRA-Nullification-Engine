# Evolutionary Roadmap Module

## Обзор
Модуль предсказания эволюции потребностей на основе GRA-обнуления.

## API
- `GET /api/roadmap/{user_id}` — персональная дорожная карта.
- `POST /api/roadmap/feedback` — обратная связь для обучения.
- `GET /api/aggregated/demand` — агрегированный прогноз (Enterprise).

## Подписки
- Premium: дорожная карта на 3 года.
- Visionary: до 10 лет + сценарные ветвления.
